WHAT IS IT?
A java program that creates 3 dimensional type maker. The maker's cycloidic generative process can be viewed stereoscopically with Google cardboard as an app for android.
IDEA
It's a cycloidic typewriter that explorates a dimensional interpolations between forms. Conclusion from the experiment: the generation of different cycloid and letter pairings needs to be more considered according to complimentary rotational nodes
EXECUTION
Java on Processing with Geomerative library and Android SDK installed. Ported program to compatible android phone on developer mode
Original code with android SDK deprecated.

  • Github documentation coming soon

MORE CONTEXT?
This particular idea actually first started as part of my breadth explorations while making a poster for a lecture series on code and form. I wanted to explore the idea of code and form more literally in my visuals by embodying it in how I generated graphics for the poster. I didn't end up using it for the final poster, but I was happy to have stretched my visual skills.
